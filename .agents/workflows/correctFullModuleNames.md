---
description: Check les full module names
---

Check les full module names présents dans le code (/src).
Ils doivent être égaux au module en 1e ligne.

E.g. 

module Inter.Ui.Router.Menu.UnfoldIcon.Style where

import ...

fullModuleName :: String
fullModuleName = "Inter.Ui.Router.Menu.UnfoldIcon.Style"

...
